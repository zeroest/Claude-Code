
# 토큰을 효과적으로 사용하는 방법

## 1. 프롬프트로 토큰 절약하기

특히 "나만의 프로필 사이트" 미션을 진행하실 때, 프롬프트를 어떻게 작성하느냐에 따라 토큰 사용량이 크게 달라집니다.

❌ 토큰을 많이 사용하는 프롬프트

```
프로필 웹사이트 만들어줘
```

이렇게 간단하게 요청하면 Claude가 다음과 같이 과도하게 생성할 수 있습니다:

- 복잡한 HTML 코드
- 여러 개의 분리된 파일 (HTML, CSS, JS, Python 등)
- 배포 환경 설정까지 포함
- 불필요한 최적화 및 테스트 코드

✅ 토큰을 절약하는 프롬프트

```
간단한 프로필 웹사이트를 만들어줘:
- 단일 HTML 파일로만 작성
- 이름, 소개, SNS 링크만 포함
- 외부 라이브러리 사용하지 말고 순수 HTML/CSS만 사용
- 배포, 테스트, 최적화 단계는 제외
- 총 300줄 이하로 작성
```

핵심 키워드

- "간단한", "최소한의 기능만"
- "단일 파일로"
- "배포/테스트/최적화 제외"
- "~줄 이하로 작성"

범위를 명확히 제한하면 Claude가 꼭 필요한 코드만 생성하여 토큰을 크게 절약할 수 있습니다! 프로필 웹사이트 만들기 뿐만 아니라 다양한 미션을 진행할때 프롬프트로 범위를 명확히 제한해주면 좋아요.

## 2. Haiku 모델 활용하기
강의 학습 단계에서는 /model 명령어를 사용해 Haiku 모델로 전환하는 것을 적극 추천드립니다!

Haiku 모델의 장점

- ⚡ 빠른 응답 속도: Sonnet 대비 훨씬 빠른 실행
- 💰 적은 토큰 사용: 같은 작업도 더 효율적으로 처리
- 📚 학습에 최적: 기본 개념 학습과 간단한 실습에 충분한 성능

사용 방법

```
/model haiku
```

터미널에서 위 명령어를 입력하면 즉시 Haiku 모델로 전환됩니다.

언제 사용하면 좋을까요?

- ✅ 강의 미션 실습할 때
- ✅ 간단한 앱이나 기능 구현할 때
- ✅ 코드 수정이나 디버깅할 때
- ✅ 기본 개념을 학습할 때

특히 Pro 플랜 사용자분들께 권장드립니다! Pro 플랜은 충분한 사용량을 제공하지만, Haiku를 활용하면 더 많은 실습을 진행하실 수 있고, 5시간 리셋 주기를 더 여유롭게 사용하실 수 있습니다.

복잡한 프로젝트나 높은 품질이 필요할 때만 Sonnet을 사용하고, 학습과 실습에는 Haiku를 사용하는 전략을 추천드립니다!

## 3. 컨텍스트 관리 명령어로 토큰 최적화하기 🎯

토큰 관리의 핵심은 컨텍스트 관리입니다. 뒤에서 자세히 다루는 내용이지만, 미리 알고 계시면 강의를 훨씬 효율적으로 학습하실 수 있습니다.

필수 명령어 3가지

- /context: 현재 컨텍스트에 포함된 파일 확인
- /clear: 불필요한 컨텍스트 제거
- /compact: 컨텍스트 압축으로 토큰 절약
  - /compact [주요 정보를 넣으면 해당 정보를 위주로 컴팩트한다]

+ /config -> auto-compact true 설정 

왜 미리 봐야 할까요? 강의 초반부터 이 명령어들을 알고 계시면:

- 토큰 제한에 도달하는 것을 예방할 수 있습니다
- 더 많은 실습과 미션을 진행할 수 있습니다
- Claude Code의 동작 원리를 더 깊이 이해할 수 있습니다
