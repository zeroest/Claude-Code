
# 슬래시 명령어

## 🔄 **세션 관리**

**`/clear`**

- 대화 기록을 완전히 삭제해서 새로 시작
- `/clear` → 모든 기록 삭제하고 깨끗하게 시작

**`/compact [지시사항]`**

- 대화 기록을 요약해서 컨텍스트는 유지하되 토큰 절약
- `/compact 중요한 코드 변경사항만 요약해줘` → 핵심만 남기고 압축

**`/resume`**

- 과거 대화 세션을 선택해서 이어서 진행
- `/resume` → 이전 대화 목록에서 선택

**`/exit`**

- Claude Code 세션 종료
- `/exit` → 터미널로 돌아가기

## 🏗️ **프로젝트 초기화**

**`/init`**

- CLAUDE.md 파일로 프로젝트 메모리 초기화
- `/init` → 프로젝트 설명과 규칙을 CLAUDE.md에 저장

**`/add-dir`**

- 추가 작업 디렉토리를 Claude가 접근할 수 있게 설정
- `/add-dir ../libs ../docs` → 여러 폴더를 동시에 접근 가능하게

## 👤 **계정 관리**

**`/login`**

- Anthropic 계정으로 로그인
- `/login` → 브라우저에서 인증 후 로그인

**`/logout`**

- 현재 계정에서 로그아웃
- `/logout` → 계정 연결 해제

## ⚙️ **시스템 관리**

**`/doctor`**

- Claude Code 설치와 설정 상태 진단
- `/doctor` → 모든 설정이 올바른지 체크

**`/status`**

- 현재 Claude Code 상태 (버전, 모델, 계정 등) 확인
- `/status` → 전체 시스템 정보 표시

**`/config`**

- 설정 패널 열어서 각종 옵션 변경
- `/config` → GUI로 설정 변경

**`/upgrade`**

- Max 플랜으로 업그레이드 (더 높은 사용량 한도)
- `/upgrade` → 결제 페이지로 이동

**`/migrate-installer`**

- 글로벌 npm 설치에서 로컬 설치로 마이그레이션
- `/migrate-installer` → 설치 방식 변경

## 🔧 **개발 도구 연동**

**`/ide`**

- IDE 통합 관리 및 상태 확인
- `/ide` → VS Code나 JetBrains 연결 상태 체크

**`/mcp`**

- Model Context Protocol 서버 관리
- `/mcp add github-server` → GitHub MCP 서버 추가

**`/terminal-setup`**

- Shift+Enter로 줄바꿈 키바인딩 설치
- `/terminal-setup` → 터미널에서 편리한 입력을 위한 설정

**`/statusline`**

- Claude Code 상태표시줄 UI 설정
- `/statusline` → 터미널에 상태정보 표시

**`/vim`**

- Vim 편집 모드와 일반 모드 전환
- `/vim` → Vim 키바인딩 활성화/비활성화

## 🐙 **GitHub 관련**

**`/install-github-app`**

- 저장소에 Claude GitHub Actions 설정
- `/install-github-app` → CI/CD에 Claude 통합

**`/pr-comments`**

- GitHub 풀리퀘스트 댓글 가져오기
- `/pr-comments #123` → PR 번호로 댓글 확인

**`/review`**

- 풀리퀘스트 리뷰 진행
- `/review #456` → 특정 PR 코드 리뷰

**`/security-review`**

- 현재 브랜치의 보안 리뷰 수행
- `/security-review` → 보안 취약점 검사

## 🛡️ **권한 및 설정**

**`/permissions`**

- 도구 권한 허용/거부 규칙 관리
- `/permissions` → 파일 수정 권한 등 설정

**`/hooks`**

- 도구 이벤트에 대한 훅 설정 관리
- `/hooks` → 파일 저장 시 자동 실행할 스크립트 설정

**`/memory`**

- Claude 메모리 파일(CLAUDE.md) 편집
- `/memory` → 프로젝트 규칙과 컨텍스트 수정

## ℹ️ **정보 및 도움**

**`/help`**

- 사용 가능한 명령어 도움말 표시
- `/help` → 모든 명령어 목록과 설명

**`/cost`**

- 현재 세션의 총 비용과 소요시간 표시
- `/cost` → 토큰 사용량과 API 비용 확인

**`/release-notes`**

- Claude Code 릴리스 노트 확인
- `/release-notes` → 최신 업데이트 내용

**`/bug`**

- Claude Code 피드백 및 버그 리포트 제출
- `/bug` → 현재 대화를 Anthropic에 피드백으로 전송

## 📊 **데이터 관리**

**`/export`**

- 현재 대화를 파일이나 클립보드로 내보내기
- `/export conversation.md` → 대화 내용을 마크다운으로 저장

**`/agents`**

- 에이전트 설정 관리
- `/agents` → 서브에이전트 구성 및 관리

**`/bashes`**

- 백그라운드 bash 쉘 목록 및 관리
- `/bashes` → 실행 중인 백그라운드 프로세스 확인

## 🤖 **AI 모델**

**`/model`**

- Claude Code에서 사용할 AI 모델 선택
- `/model` → Sonnet, Opus 등 모델 변경

---

## 💡 **사용 팁**

- **자주 사용**: `/help`, `/status`, `/clear`, `/compact`
- **프로젝트 시작**: `/init`, `/memory`, `/config`
- **문제 해결**: `/doctor`, `/bug`, `/cost`
- **고급 기능**: `/mcp`, `/hooks`, `/agents`
