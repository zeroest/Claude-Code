
# Settings.json

https://code.claude.com/docs/ko/settings

| 기능 | 사용자 위치 | 프로젝트 위치 | 로컬 위치 |
|------|------------|-------------|----------|
| Settings | `~/.claude/settings.json` | `.claude/settings.json` | `.claude/settings.local.json` |
| Subagents | `~/.claude/agents/` | `.claude/agents/` | — |
| MCP servers | `~/.claude.json` | `.mcp.json` | `~/.claude.json` (프로젝트별) |
| Plugins | `~/.claude/settings.json` | `.claude/settings.json` | `.claude/settings.local.json` |
| CLAUDE.md | `~/.claude/CLAUDE.md` | `CLAUDE.md` 또는 `.claude/CLAUDE.md` | `CLAUDE.local.md` |

- 사용자 위치: 모든 프로젝트에 적용되는 전역 설정
- 프로젝트 위치: 저장소에 커밋되어 팀과 공유됨
- 로컬 위치: 해당 프로젝트에서만 사용자 본인에게 적용 (gitignore 처리 할 것)

## 설정 예시

- defaultMode: "plan"
- additionalDirectories: ["../remote_project"]
- deny: ["Read(./.env)", "Read(./secrets/**)"]
